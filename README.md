A browser remake of the original Gwent minigame from The Witcher 3: Wild Hunt including all cards from the DLC.<br/>Click [here](https://renanb4.github.io/gwent-classic/) to play.<br>
In addition to cloning the original game, various cards, faction, abilities and features have added to push the game even further.<br>

## Operations included with the keyboard:

**"E"** starts the game<br />
**"X"** uses or modifies the leader card<br />
**"Q"** closes the card explanation windows or gives up on the game<br />
**"Space"** passes the round<br />
**"Enter"** plays the cards<br />
**Arrows** select cards on the carousel
